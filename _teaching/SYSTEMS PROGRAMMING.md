Systems Programming focuses on the development of software that interacts closely with computer hardware and operating systems. Students will learn to write efficient, low-level code using languages such as C and assembly. Key topics include memory management, process control, file systems, and concurrency. Practical assignments and projects will reinforce concepts by implementing real-world systems-level applications. This course prepares students for advanced topics in computer science and careers in systems software development.

Memory Management and Error Detection
======
In this course, students will explore systems programming by implementing fundamental memory management functions such as `malloc()` and `free()`. Emphasis is placed on understanding heap memory, managing memory allocation, and detecting common usage errors. Through practical coding assignments, students will develop low-level programming skills, learn to create and test custom libraries, and gain experience in handling dynamic memory in C. This course provides essential knowledge for designing efficient systems software and debugging complex memory issues.

File I/O and Spell-Checking
======

This course immerses students in systems programming by developing a spell-checking program that reads dictionary files and verifies text file accuracy. Students will learn to handle file I/O operations, directory traversal, and efficient data structures for quick lookups. The focus is on using Posix functions, managing text processing, and ensuring robust error detection. Through hands-on projects, students gain practical experience in building and testing low-level software, emphasizing code modularity, performance, and error handling.

Command-Line Shell Implementation
======
This course delves into systems programming by creating a custom command-line shell similar to bash. Students will gain experience with Posix stream I/O, managing working directories, spawning child processes, and handling pipelines and conditionals. The course emphasizes implementing features such as interactive and batch modes, command parsing, input/output redirection, and wildcard expansion. Through building a functional shell, students will enhance their understanding of process control, inter-process communication, and low-level programming in C. This practical approach equips students with essential skills for systems software development.
